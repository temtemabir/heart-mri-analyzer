# ❤️ Heart MRI Analyzer

Heart MRI Analyzer is a mobile application developed as a Final Year Project in Software Engineering. The application assists in analyzing cardiac MRI images using a Machine Learning model based on the **Random Forest** algorithm.

It combines mobile development, cloud services, and machine learning to provide a simple and efficient solution for medical image analysis.

---

## 📱 Features

- Secure user authentication
- MRI image upload
- Automated image analysis
- AI-based prediction using Random Forest
- Patient information management
- Doctor and patient roles
- User-friendly interface
